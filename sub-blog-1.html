VMware DRS

ဟုတ်ကဲ့ အားလုံးပဲ မင်္ဂလာပါခင်ဗျာ🙏။ itGateway Technology Group ရဲ့ Knowledge Sharing Section လေးကနေ ပြန်လည် ကြိုဆိုလိုက်ပါတယ်👋။
✍ဒီတစ်ခေါက် ပြောပြပေးသွားမယ့် အကြောင်းအရာလေးကတော့ လူသိများတဲ့ အကြောင်းရာတွေထဲကတစ်ခု ဖြစ်တဲ့ VMware DRS Feature လေးအကြောင်းဖြစ်ပါ။

What is VMware DRS??
VMware DRS ဆိုတာဘာလဲ??သူကဘယ်လိုအလုပ်လုပ်သလဲပေါ့??
DRS ရဲ့အရှည်ကတော့ Distributed Resource Schedular ပါခင်ဗျာ။DRS ဘယ်လိုအလုပ်လုပ်လဲ ဆိုတော့သူ့နာမည်အတိုင်းပဲလေ ESXI host ရဲ့ Resource (CPU,Memory,etc…….) စတဲ့တွေကိုမျှဝေပေးတာပေါ့။ဘယ်လိုမျှဝေပေးလဲဆိုတာပေါ့ အဲ့တာကမှကျွန်တော်တို့အဓိကသိရမှာပေါ့။

VMware DRS ဆိုတာကတော့ Cluster ထဲက host တွေပေါ်မှာရှိတဲ့ VM တွေရဲ့လိုအပ်တဲ့ resource တွေကို balanced ဖြစ်အောင် auto ချိန်ညှိပေးပြီး Cluster ရဲ့ Performance ကိုထိန်းညှိပေးတာပေါ့။

How does DRS work???
DRS ဘယ်လိုအလုပ်လုပ်လဲပေါ့။ကျွန်တော်တို့ Cluster မှာ DRS ကို Enabled လုပ်လိုက်ပြီဆိုတာနဲ့ ကျွန်တော်တို့က VM တစ်လုံးကို ဘယ် ESXI host ပေါ်မှာ Power-on သင့်တယ်ဆိုတာကို အကြံပေးတယ်။သေချာရှင်းပြရမယ်ဆိုရင် အရင်ဆုံး DRS algorithm က ဘယ် ESXI host က VM အသစ်အတွက် အကောင်းဆုံးဖြစ်မလဲဆိုတာ ဆုံးဖြတ်ပေးတယ်။ပြီးတော့ DRS algorithm က VM အသစ်ရဲ့ resource က ကျွန်တော်တို့ရဲ့ Cluster balanceတစ်ခုလုံးကို ဘယ်လို affect ဖြစ်မလဲဆိုတာ ချိန်စတယ်။နောက်ဆုံးကျမှ VM ကို Power-on လိုက်ပြီး DRS Monitoring Pool ထဲကိုထည့်လိုက်တာပေါ့။

DRS ရဲ့ Main Goal ကဘာလဲပေါ့????
DRS ရဲ့ Main Goal ကကျွန်တော်တို့ Cluster ကို balanced ဖြစ်အောင်ထားပေးပြီးတော့ Cluster ထဲမှာရှိတဲ့ VM တွေ run နေတဲ့အချိန် သူတို့ လိုအပ်တဲ့ resource ကိုလုံလုံလောက်လောက်ရရှိအောင်လုပ်ဆောင်ပေးတာပေါ့။DRS ကနေပြီးတော့ Cluster balanced ဖြစ်အောင် host တွေနဲ့ VM တွေရဲ့ resource usage တွေကို 5 minute တစ်ကြိမ် health check လုပ်တယ်။အကယ်၍ DRS algorithm က Cluster ထဲက usage တွေ သိသိသာသာပြောင်းလဲသွားတယ်လို့ ဆုံးဖြတ်ရင် vMotion လို့ခေါ်တဲ့ VM တစ်ခု ဒါမှမဟုတ် VM အများစုကို တိုက်ရိုက်‌‌ရွှေ့ပြောင်းခြင်းကိုစတင်တာပေါ့။vMotion လုပ်ဆောင်နေတဲ့အချိန်မှာ VM တွေကတော့ active ဖြစ်နေပြီးပုံမှန်အတိုင်းပဲ သုံးလို့ရတာပေါ့။vMotion မတင်ခင် DRS ကနေပြီး‌ရွေးချယ်ထားတဲ့ Source ESXI host နဲ့ Destination ESXI host တွေမှာတော့ Performance တွေကိုထိခိုက်မှာပေါ့။VMware DRS ကတော့ ကျွန်တာ်တို့ Cluster ရဲ့ performance နဲ့ stability ကို ထိန်းသိမ်းပေးပြီး VM တွေကို Manage လုပ်ပေးတဲ့ software တစ်ခုပေါ့။
